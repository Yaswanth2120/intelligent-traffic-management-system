# Phase 6

Phase 6 moves the platform from local orchestration to deployable cloud-native assets.

## Part 1: Kubernetes Baseline

- Namespace and shared configuration
- Deployments and services for all core services
- Operational environment variables for Kafka, Redis, PostgreSQL, and Prometheus

## Part 2: Autoscaling And Runtime Policies

- Horizontal Pod Autoscalers for all runtime services
- CPU-based scaling as baseline
- Annotation and labeling structure for future traffic/Kafka-lag scaling
